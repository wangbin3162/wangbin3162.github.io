# 一、Sticky footer布局

如果页面内容不够长的时候，页脚块粘贴在视窗底部；如果内容足够长时，页脚块会被内容向下推送

<div class="main-wrapper">
    <div class="auto-wrapper"></div>
</div>
<div class="footer"></div>

.auto-wrapper {
    padding-bottom: 64px;
}
.footer {
    position: relative;
    margin: -64px auto 0;
}
