# 介绍

上一个标准版本 ES5 在 2009 年发布以后，应用非常广泛，甚至在新的新 ECMAScript 标准 (ES6或ES2015)发布以后仍然占据绝大部分的开发，本章节 记录并整理部分工作中常见的或应用技巧的文档。

ES6发布以来，一直以新语法、新特性的优越性吸引著众多 JavaScript 开发者，驱使他们积极的进行开发和学习，这里会就部分ES6新特性进行记录整理， 以便今后的工作学习。
