## 开源库汇总

GitHub地址均为 https://github.com/wangbin3162/+ '库名称'.

演示地址均为 https://wangbin3162.github.io/+ '库名称'

| 库名称 | 技术栈 | 一句话描述 |
| --- | --- | --- |
| bin-ui-design | Vue3 + TypeScript | 一个基于 Vue3 和 TypeScript 的组件库 |
| bin-admin-pro | Vue3 全家桶 | 基于bin-ui 的后端管理系统 |
| bin-editor-next | Vue3 + brace | 基于 brace 的 Vue3 编辑器组件库 |
| bin-datav | Vue3 + TypeScript | 一个基于 Vue3 和 TypeScript 的数据可视化组件库 |
| bin-datav-schema | Vue3 + TypeScript | vite + vue3 + bin-ui-design + bin-datav的数据可视化大屏框架 |
| bin-grid-layout | Vue3 + TypeScript | 一个基于 Vue3 和 TypeScript 的网格布局组件库 |
| bin-charts-next | Vue3 + echarts | 一个基于 Vue3 和 echarts 的图表组件库 |
| bin-form-maker | Vue3 + TypeScript | 一个基于 Vue3 和 TypeScript 的表单生成器组件库 |
| bin-excel-pro | Vue3 + univer | 一个基于 Vue3 和 univer 的 Excel 表格组件库 |
| bin-tree-org | Vue2 | 一个基于 Vue2 的组织树组件 |
| bin-animation | Vue | 基于vue，结合transition钩子函数配合的css3动画库 |
| bin-keyframe-animation | javascript | js关键帧动画库 |
